K-Clock
PC용 플로팅 디지털 시계
v1.3.8 2026-07-24 ⬇ 설치형 다운로드
항상 위에 고정 안정성 개선, 새 아이콘 디자인 적용, 프로그램 정보 창 새 디자인, 프로그램 제거 확인 화면 개선.

📦 설치 방법

  1. 위 다운로드 버튼 클릭 → K-Clock_x64-setup.exe 다운로드
  2. exe 파일 실행
  3. Windows 보호 경고가 뜨면 "추가 정보" 클릭 후 "실행"
  4. 설치 완료 후 바탕화면에서 K-Clock 실행

주요 기능

🪟 투명 배경바탕화면이 그대로 보이는 투명 시계
📌 항상 위다른 창 위에 항상 표시
⏰ 알람시간+이름 설정, 활성화 토글, 시계 깜빡임 알림
⏱ 카운트다운시/분/초 타이머, 시작·일시정지·초기화
🔤 폰트 8종KoPubWorld, 맑은 고딕, 굴림, 나눔고딕 등
🎨 밝기·크기슬라이더로 세밀하게 조정

사용 방법

  • 메뉴 열기 — 시계 위에서 우클릭
  • 위치 이동 — 시계 드래그
  • 알람 설정 — 우클릭 → 알람 → 시간·이름 입력 후 추가
  • 카운트다운 — 우클릭 → 카운트다운 → 시/분/초 입력 후 시작
  • 폰트·크기·밝기 — 우클릭 → 설정
